# ApiVideo

Práctica 5 Api HTML5

Programa principal practica.html. Se ha pintado un pequeño canvas donde se pone el título de la práctica.

Hay un botón que nos permite seleccionar el archivo a reproducir del tipo mp4 que está ubicado en la misma ruta que nuestro fichero .html

Una vez seleccionado el video a reproducir se inserta en la página de forma dinámica junto con los botones especificados.

Se implementan los mensajes solicitados mientras dura la carga del archivo.
